# VS Code Productivity Shortcuts

Ctrl + Shoft + P - search for commands/shortcuts
Ctrl + D - multiselection of words/variables, press several times to select next word
Alt + Click - mulyicursor
Alt + arrows up/down - to move lines
Ctrl + / - comment line
Alt + Shift + F - use Formater
Ctrl + P - top search bar for files etc
F12 - after clicking on the function click F12 to go to its deffinition
Ctrl + B - hide/show left sidebar
Ctrl + ` - open/select terminal
F2 - select variable and click F2, change its name everywhere
Ctrl \ - split screen
Ctrl + space - hint
Ctrl + Shift + K - delete full row
